## About the Project
NewsOS is an AI-powered platform that analyzes and summarizes financial news in real-time.  
It helps users quickly understand complex market information and make smarter decisions without information overload.

## Problem Statement
Every day, users are overwhelmed by large volumes of financial news across multiple platforms.  
Understanding relevant insights quickly is difficult and time-consuming.

## Solution
NewsOS uses AI to:
- Extract key insights from news articles  
- Summarize complex information instantly  
- Deliver clear, concise, and actionable content  

## Key Features
- AI-powered financial news summarization  
- Real-time news processing  
- Multi-language capability  
- Insight-driven analysis  
- Clean and simple user experience

## Dashboard Sections
The NewsOS dashboard consists of the following core sections:
- My Newsroom  
  Central hub displaying curated and categorized financial news cards  
- Navigator  
  Helps users explore and filter news based on topics or categories  
- Video Studio  
  Converts news insights into video-ready content or scripts  
- Story Arc  
  Structures news into a narrative format for better understanding  
- Vernacular  
  Enables multi-language support for regional accessibility    
- News Cards / Feed  
  Visual display of summarized articles with key insights  
- Insight Panel  
  Shows extracted key points and analysis from selected news  

## How It Works
1. News data is collected from relevant sources  
2. AI processes and filters important information  
3. Key insights are extracted and summarized  
4. Users receive clear, structured outputs

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- AI Integration: Large Language Models  
- Deployment: Vercel  

##Run Locally

*Prerequisites:* Node.js  

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
